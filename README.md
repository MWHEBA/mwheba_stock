# مخازن موهبة - MWHEBA Stock

نظام متكامل لإدارة المخزون والحسابات المالية مبني باستخدام Django.

## الميزات الرئيسية

### إدارة المشتريات
- تسجيل فواتير الموردين
- تحديث المخزون تلقائيًا عند الشراء
- حساب التكلفة الإجمالية لكل عملية شراء

### إدارة المبيعات
- إنشاء فواتير للعملاء
- متابعة عمليات الدفع
- تخصيص خصومات وعروض خاصة

### إدارة المخزون
- تتبع الكميات المتاحة من المنتجات
- إدارة مواقع التخزين المختلفة
- تنبيهات عند انخفاض المخزون
- إمكانية تحديد هوامش الربح للمنتجات

### إدارة العملاء والموردين
- إضافة وتحديث بيانات العملاء والموردين
- سجل التعاملات لكل عميل أو مورد
- إمكانية إضافة ملاحظات لكل مورد أو عميل

### إدارة الحسابات
- تتبع الإيرادات والمصروفات
- حساب الأرباح والخسائر
- إدارة الأصول والخصوم

### التقارير المالية والتفصيلية
- تقارير مبيعات ومشتريات مفصلة
- تقارير الأرباح والخسائر مع الرسوم البيانية التوضيحية
- إمكانية تصدير التقارير إلى CSV أو PDF
- تقارير محدثة بشكل دوري (يومي/أسبوعي/شهري)

### إدارة بيانات الشركة
- تخصيص بيانات الشركة والشعار
- تخصيص الفواتير والتقارير الخاصة بالشركة

## التقنيات المستخدمة

- **الواجهة الخلفية (Backend)**: Python (Django)
- **قاعدة البيانات**: SQLite (افتراضي) أو PostgreSQL (اختياري)
- **الواجهة الأمامية (Frontend)**: Bootstrap + JavaScript + HTML/CSS
- **التحليل المالي**: رسوم بيانية توضح تدفق الإيرادات، المصروفات، وأرباح كل منتج باستخدام Chart.js

## متطلبات النظام

- Python 3.10 أو أحدث
- pip (مدير حزم Python)
- PostgreSQL (اختياري)

## خطوات التثبيت

### 1. إعداد بيئة Python الافتراضية

```bash
# إنشاء بيئة افتراضية
python -m venv venv

# تفعيل البيئة الافتراضية
# في Windows
venv\Scripts\activate
# في Linux/Mac
source venv/bin/activate
```

### 3. تثبيت المكتبات المطلوبة

```bash
pip install -r requirements.txt
```

### 3. إعداد قاعدة البيانات

النظام يستخدم SQLite افتراضيًا، وهي قاعدة بيانات لا تتطلب إعدادًا إضافيًا.

قم بتشغيل سكريبت إعداد قاعدة البيانات:

```bash
python setup_database.py
```

#### استخدام PostgreSQL (اختياري)

إذا كنت ترغب في استخدام PostgreSQL بدلاً من SQLite، قم بتثبيت PostgreSQL من [الموقع الرسمي](https://www.postgresql.org/download/) وإنشاء مستخدم وكلمة مرور، ثم قم بتعديل ملف `core/settings.py`:

1. قم بإلغاء التعليق على إعدادات PostgreSQL وتعليق إعدادات SQLite:

```python
# DATABASES = {
#     'default': {
#         'ENGINE': 'django.db.backends.sqlite3',
#         'NAME': BASE_DIR / 'db.sqlite3',
#     }
# }

DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.postgresql',
        'NAME': 'mwheba_stock',
        'USER': 'postgres',  # قم بتغييره إلى اسم المستخدم الخاص بك
        'PASSWORD': 'postgres',  # قم بتغييره إلى كلمة المرور الخاصة بك
        'HOST': 'localhost',
        'PORT': '5432',
    }
}
```

2. قم بإلغاء التعليق على حزمة psycopg2 في ملف requirements.txt وتثبيتها:

```bash
pip install psycopg2-binary
```

### 5. تشغيل الخادم

```bash
python manage.py runserver
```

يمكنك الآن الوصول إلى التطبيق من خلال المتصفح على العنوان: http://127.0.0.1:8000/

## تسجيل الدخول

بعد إعداد قاعدة البيانات، يمكنك تسجيل الدخول باستخدام:

- **اسم المستخدم**: admin
- **كلمة المرور**: admin123

## الاستخدام في بيئة الإنتاج

للاستخدام في بيئة الإنتاج، يجب تعديل الإعدادات التالية في ملف `core/settings.py`:

```python
DEBUG = False
ALLOWED_HOSTS = ['yourdomain.com', 'www.yourdomain.com']

# إعدادات الأمان
SESSION_COOKIE_SECURE = True
CSRF_COOKIE_SECURE = True
SECURE_SSL_REDIRECT = True
```
